This class is the basic class which is used to make a request to Tumblr API.
This class has two variables, which are RequestBuilder to perform the actual request and Stirng that store the apikey.

This class has three constructors.
The default constructor generates RequestBuilder newly and joins this JumblrClient together. This is called from other constructors inside.
The second constructor takes consumerKey and consumerSecret. These two arguments are set in RequestBuilder. consumerKey is set as apikey.
The third constructor takes consumerKey, consumerSecret, token and tokenSecret. The consumerKey and consumerSecret are used for calling the second constructer, and token and tokenSecret are set by setToken method.
